# Google Sheets Setup (29 BIS)

## 1) Create a new spreadsheet
Suggested name: `29BIS - Pedidos y Precios`

## 2) Import the CSV templates as tabs
In Google Sheets: `Archivo > Importar > Subir`.

Import each file as a **new sheet**:
- `orders_template.csv` -> rename sheet to `orders`
- `prices_template.csv` -> rename sheet to `prices`
- `meta_template.csv` -> rename sheet to `meta`

## 3) Add dropdowns in `orders`
Set data validation for these columns:

- `payment_status` (column `T`):
  - `Pendiente`
  - `Pagado`

- `production_status` (column `U`):
  - `Recibido`
  - `En revisión`
  - `En producción`
  - `Listo para retirar`
  - `Entregado`
  - `Cancelado`

Tip: you can reference `meta!B:B` filtered by key, or set these lists manually.

## 4) What each tab is for
- `orders`: all incoming orders from the form
- `prices`: editable prices + `active` stock flag (`TRUE`/`FALSE`)
- `meta`: status catalogs

## 5) Notes
- `order_number` should be generated by Apps Script on insert.
- `file_url` should be filled after file upload integration to Drive.
- In `prices`, set `active = FALSE` to disable papers/specific entries.
